The project

Within this project, the goal is to measure and visualize the network topology for a selected set of host nodes. This way the idea is to get a good overview and understanding of how ASes are connected to each other and how routing agreements influence the paths network traffic takes from source to destination.

In order to achieve this goal, we make use of the tools ping and traceroute, which we run with a certain list of hosts to collect our data. Afterwards, we use the libraries Chart.js and OpenLayers in order to visualize the gathered data. We learn how the standardized exterior protocol of BGP - the Border Gateway Protocol - exchanges routing and reachability information among autonomous systems on the Internet.

How to run it

  1. Clone the repository: git clone repository-name
  2. Install requirements: pip3 install requirments.txt
  3. Two things need to run at the same time:
    1. To run the server: python3 manage.py runserver
    2. To run the background tasks:
      • python3 manage.py makemigrations
      • python3 manage.py migrate --run-syncdb
      • python3 manage.py process_tasks

By running the commands mentioned above, this project will run and you will be able to use the different features it provides. Please click the 'Get Started!' button in order to direct to the pages for data collection - the ping and traceroute - or the pages for data visualization - showing in charts, graphs and on a map. The interface of the different features should be intuitive enough to understand how you are able to use them without any additional explanation.
Have fun watching the different data and playing around with our network visualization tool!
